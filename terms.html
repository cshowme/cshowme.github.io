一、服務說明

1.1 服務內容

本服務為學術文獻 DOI（Digital Object Identifier）驗證工具，提供以下功能：

  • 驗證 DOI 編號的真實性與有效性
  • 查詢文獻書目資訊（標題、作者、期刊、年份等）
  • 偵測可能由 AI 生成之虛構文獻引用
  • 匯出驗證結果（RIS、BibTeX、CSV 等格式）
  • 付費版額外功能：雙引擎驗證（CrossRef + OpenAlex）、SJR 排名查詢、撤稿偵測、引用分析、開放取用偵測

1.2 服務版本

免費版免費CrossRef 單引擎驗證、RIS 匯出、每次最多 10 筆 DOI
Pro 版依方案定價全功能、雙引擎驗證、每次最多 50 筆 DOI
Enterprise 版專案報價全功能 + API 存取 + 多人管理

七、免責聲明

重要提醒：以下免責聲明為本服務條款之重要組成部分，請詳細閱讀。

7.1 驗證結果之準確性

本服務之驗證結果係依據第三方學術資料庫（CrossRef、OpenAlex、DataCite 等）之回傳資料產生。本公司不保證驗證結果 100% 正確，包括但不限於：

  • 第三方資料庫的資料可能有延遲、錯誤或遺漏
  • 新出版之文獻可能尚未被資料庫收錄
  • DOI 狀態可能因出版商操作而變動
  • 撤稿偵測結果可能有誤判或漏判

驗證結果僅供參考，使用者應自行判斷並確認文獻之真實性。本服務不構成學術上的最終認定。
