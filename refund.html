一、數位服務性質說明

本服務「AI 文獻品管師 DOI Verifier Pro」提供即時線上學術文獻 DOI 驗證功能，屬於「非以有形媒介提供之數位內容或一經提供即為完成之線上服務」。

依據《消費者保護法》第 19 條第 1 項但書及行政院《通訊交易解除權合理例外情事適用準則》第 2 條第 5 款規定，經消費者事先同意並開始使用服務後，不適用七日猶豫期（鑑賞期）。

購買前請先試用：本服務提供免費版，您可以在購買前充分體驗核心驗證功能。亦可參考教學影片了解完整功能。

二、退款原則

本服務為數位訂閱服務，付款啟用後原則上不提供退款。
此做法與 KKBOX、PressPlay、Claude Pro 等數位訂閱服務一致。

當您完成付款並開始使用服務時，視為您已：

  1. 閱讀並同意本退款政策
  2. 了解本服務為即時提供之數位內容
  3. 同意依法不適用七日猶豫期
  4. 確認已有機會透過免費版試用本服務之核心功能

三、例外退款情形

以下情況本公司將主動處理，無需額外申請：

3.1 系統重大故障

若因本公司系統故障導致服務連續 72 小時以上完全無法使用（排除第三方 API 暫時性中斷），我們將：

  • 優先方案：主動延長您的方案有效期（等同故障天數）
  • 替代方案：依您的選擇，按比例退還故障期間的費用

3.2 重複扣款

若因系統錯誤導致同一筆訂單重複扣款，我們將於確認後 5 個工作天內全額退還重複扣款金額。

3.3 服務內容重大不符

若本服務之實際功能與官方網站說明有重大差異（非因第三方 API 變更所致），且經本公司確認屬實，您可以申請退款。

四、不予退款之情況

  1. 個人因素：使用後不滿意、使用頻率低於預期、不再需要等個人原因
  2. 未取消訂閱導致的自動續約扣款：請在到期前自行取消
  3. 使用者端問題：因使用者自身網路環境、瀏覽器相容性等導致無法使用
  4. 第三方 API 暫時中斷：CrossRef、OpenAlex 等學術資料庫偶爾之暫時性中斷，非本公司可控
  5. 違反服務條款：因違反使用規範被停權或終止帳號者

五、取消訂閱

5.1 如何取消

您可隨時透過會員中心或聯繫客服取消訂閱。

5.2 取消後的權益

  • 取消後，您的付費方案仍維持有效至原定到期日
  • 到期後不再自動扣款
  • 到期後帳號轉為免費版，已驗證的歷史紀錄保留 30 天供您匯出
  • 取消訂閱不等同退款，剩餘天數的費用不會退還

六、反濫用條款

本公司保留審核退款申請之權利。若發現使用者有以下行為，得拒絕退款申請：

  1. 反覆訂購後取消並申請退款
  2. 大量使用驗證功能後立即申請退款
  3. 使用行為明顯異常（如短時間內大量查詢後即取消）
  4. 其他有濫用退款機制之虞的行為
